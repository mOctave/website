mOctave

Hi! I'm mOctave, an amateur Java developer, Endless Sky reviewer, and ranker of flags. Oh, and I seem to be doing way too much website design these days.

Here's a couple of things I've done. For a fuller (but still incomplete) list, see my projects page.

Flagrank

Have you ever felt personally offended by a bad flag design? Sat down and spent hours of your life trying to build a tier flaglist? Watched that one CGP Grey video on loop? Well, here's the site for you. Pit flags against each other in an infinite series of head-to-head matches and let your (clearly superior) opinions on vexillology influence the flags' online Elo ratings in real time.

Weftspace

Weftspace is a library designed to offer the maximum possible level of support for Endless Sky datafile syntax. Available and maintained for Java and Python, and useful for providing datafile support for lightweight apps where the structure of a JSON or YAML file isn't quite what you're looking for.

Blended Ships

Blended Ships is by far my largest and most polished ES plugin, attempting to introduce some element of realism (sorry Azure) into ship splicing.

Eratosthenes Benchmark

A 2024 project of mine where I tried to benchmark a number of programming languages and learn a bit more about them in the process. As far as I'm aware, all the code there still should work, and you're welcome to PR another language. However, I'm not actively working on this anymore.
